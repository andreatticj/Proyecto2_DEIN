ALUMNO

Nombre

El nombre de una persona, utilizado para identificarla de manera individual. En el contexto de un alumno, sería el nombre propio del estudiante.

Dni

Un número único asignado a un individuo por la autoridad gubernamental de un país para identificación y control de ciudadanía. En el contexto de un alumno, se refiere al número de identificación nacional del estudiante.

Primer Apellido

El primer componente del apellido de una persona. En algunas culturas, el apellido se compone de dos partes, y el primer apellido se coloca antes del segundo apellido.

Segundo Apellido

El segundo componente del apellido de una persona. En algunas culturas, el apellido se compone de dos partes, y el segundo apellido se coloca después del primer apellido.
