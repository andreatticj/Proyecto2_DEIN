Préstamo

Id del Préstamo

Es un código único que identifica de manera exclusiva cada transacción de préstamo. En este caso, se asocia a un libro prestado a un usuario, permitiendo su seguimiento dentro del sistema.

DNI del Alumno

Es el número de identificación oficial de cada estudiante, asignado por las autoridades competentes. Se utiliza para asociar el préstamo al alumno que lo realiza, garantizando su trazabilidad.

Código del Libro

Es un identificador único asignado a cada libro. Este código se utiliza para registrar y gestionar el préstamo del libro dentro del sistema, evitando confusiones o duplicidades.

Fecha de Préstamo

Es la fecha en la que el libro fue retirado del sistema o biblioteca. Indica el inicio del préstamo y se utiliza para controlar la duración del mismo y la fecha de devolución.
