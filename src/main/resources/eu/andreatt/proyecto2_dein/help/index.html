Sistema de Préstamos de Libros

Datos del Alumno

Atributos: DNI, Nombre, Apellidos

El estudiante que solicita un préstamo de libro en la biblioteca.

Información del Libro

Atributos: Código, Título, Autor, Editorial, Estado

Cada libro tiene un identificador único en el sistema y está disponible para préstamo.

Registro de Préstamos

Atributos: ID de Préstamo, DNI del Alumno, Código del Libro, Fecha de Préstamo

Registra cada vez que un estudiante retira un libro.

Histórico de Préstamos o Devoluciones de Préstamos

Atributos: ID de Préstamo, DNI del Alumno, Código del Libro, Fecha de Préstamo, Fecha de Devolución

Lista detallada de todos los préstamos y devoluciones realizados en la biblioteca.
