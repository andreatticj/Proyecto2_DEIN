PRÉSTAMO DE LIBROS

Alumno

ATRIBUTOS: Dni, Nombre, Primer Apellido, Segundo Apellido

Un alumno es el usuario que realiza el préstamo de uno o varios libros.

Libro

ATRIBUTOS: Código, Título, Autor, Editorial, Estado, Baja, Portada

Un libro es prestado a uno o varios alumnos.

Préstamo

ATRIBUTOS: Id_prestamo, Dni_Alumno, Código_Libro, Fecha Préstamo

Un préstamo se realiza entre un alumno y un libro, pudiendo hacer varios préstamos simultaneamente.

Histórico de Préstamos

ATRIBUTOS: Id_prestamo, Dni_Alumno, Código_Libro, Fecha Préstamo, Fecha Devolución

El histórico de prestamos registra todos los préstamos realizados.
